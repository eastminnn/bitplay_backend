# 💵 BitPlay

> "BitPlay" 는 업비트 API를 활용한 상장된 코인으로 모의투자를 할 수 있는 웹 어플리케이션입니다.
>
> 2025.1.12 ~
> 
> 1인 프로젝트

<br>

## 🔧 Stack
- framework : Spring Boot
- language : java17
- db : MySQL
- IDE : IntelliJ, Gradle
- tool : Git, Notion, ERDCloud

<br>
<br>
  
## ⭐ Main Feature
1. 회원관리
2. 실시간 코인 시세 조회
3. 가상 자산 관리
4. 매수 / 매도 기능
5. 거래 내역 관리
